Git: El Corazón del Desarrollo Moderno

Lenguaje de Programación III • Una introducción esencial al control de versiones.

1. ¿Por qué es Vital?

Historial

Permite ver qué cambió, quién lo hizo y volver atrás si algo se rompe. Es tu red de seguridad.

Colaboración

Permite que múltiples desarrolladores trabajen en el mismo código sin pisarse los cambios.

Independencia

Puedes crear ramas para probar ideas nuevas sin afectar la estabilidad del proyecto principal.

2. Conceptos Fundamentales

Antes de escribir código, entiende cómo piensa Git:

  • Repositorio (Repo): El contenedor de tu proyecto y su historial completo.
  • Snapshot (Instantánea): Git guarda estados del proyecto, no solo cambios de líneas.
  • Working Directory: Tu área de trabajo actual (archivos modificados).
  • Staging Area: La zona de preparación donde decides qué cambios subir.
  • Local vs Remote: Tu máquina tiene una copia completa del historial, GitHub tiene otra.

3. El Kit de Supervivencia (Comandos)

Estos son los comandos que usarás el 90% del tiempo:

# Iniciar un nuevo repositorio local
git init

# Clonar un proyecto existente desde GitHub
git clone https://github.com/usuario/repo.git

# Ver el estado de tus archivos
git status

# Preparar archivos para el commit
git add .

# Guardar cambios con un mensaje
git commit -m "Explicación breve del cambio"

# Sincronizar con el servidor
git push (subir)
git pull (bajar)

4. El Flujo de Trabajo (GitHub Flow)

En el entorno profesional de Programación III, seguimos este orden:

1. Branch

Crea una rama para tu tarea: git checkout -b nueva-feature

2. Commit

Haz cambios pequeños y significativos con mensajes claros.

3. PR (Pull Request)

Pide revisión a tus compañeros antes de unir tu código a la rama main.

5. Reglas de Oro

  • No subas archivos basura: Usa siempre un archivo .gitignore.
  • Commits frecuentes: Es mejor 10 commits pequeños que uno de 500 líneas.
  • Pull antes de Push: Evita conflictos bajando los cambios de otros primero.
  • Mensajes semánticos: Usa verbos como "Corrige", "Agrega", "Elimina".
