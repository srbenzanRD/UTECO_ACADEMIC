Instalación de .NET 8

Preparando el entorno para el desarrollo con Blazor Server

SDK 8.0 VS Code VS 2022

Opción 1: Visual Studio Code

Ligero y Multiplataforma

Ideal para quienes prefieren un editor rápido y moderno. Requiere la instalación manual del SDK.

Windows Pasos a seguir:
  1. Opción Terminal (Recomendado):
    winget install microsoft.dotnet.sdk.8
  2. Opción Manual: Descarga el .NET 8 SDK x64 desde dotnet.microsoft.com. Ejecuta el instalador .exe y sigue las instrucciones.
  3. Instala VS Code.
  4. En VS Code, instala la extensión C# Dev Kit (Microsoft).
Linux (Ubuntu/Debian) Comandos:

Abre tu terminal y ejecuta:

# Instalar el SDK de .NET 8
sudo apt-get update
sudo apt-get install -y dotnet-sdk-8.0

# Verificar instalación
dotnet --version

Nota: Para otras distribuciones, consulta la documentación oficial de Microsoft.

Opción 2: Visual Studio 2022 / 2026 Community

Windows Only

Es el IDE (Entorno de Desarrollo Integrado) completo y más robusto para .NET en Windows.

Pasos para la instalación:
  1. Descarga Visual Studio Installer desde el sitio oficial.
  2. Selecciona la versión Community (Gratuita para estudiantes).
  3. IMPORTANTE: En la pestaña de "Cargas de trabajo" (Workloads), marca la casilla:
    "Desarrollo de ASP.NET y web" (ASP.NET and web development).
  4. Asegúrate de que en los componentes individuales esté marcado .NET 8.0 Runtime/SDK.
  5. Haz clic en Instalar y espera a que finalice el proceso.

✅ Verificación de la instalación

Para confirmar que todo está listo, abre una terminal (CMD, PowerShell o Terminal de Linux) y escribe:

dotnet --info

Deberías ver una lista detallada que incluya "8.0.xxx" en la sección de SDKs instalados.
