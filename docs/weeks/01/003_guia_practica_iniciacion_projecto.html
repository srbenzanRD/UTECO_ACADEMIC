Creación de Proyectos Blazor con EF Core y Autenticación

Terminal Práctica .NET 8

1 Creación del Proyecto

Para crear un proyecto Blazor que utilice el modo de renderizado Server de forma interactiva y que incluya soporte para Entity Framework Core con Autenticación Individual, ejecutamos el siguiente comando en la terminal:

dotnet new blazor -o MiProyectoAcademico -int Server -au Individual
Explicación de los parámetros:
  • -o MiProyectoAcademico: Nombre del directorio de salida.
  • -int Server: (Interactivity) Configura el modo interactivo como Server (InteractiveModeServer).
  • -au Individual: (Authentication) Configura el sistema de identidad con EF Core y una base de datos local.

2 Preparación de EF Core

Una vez creado el proyecto, debemos verificar que tenemos las herramientas de EF Core instaladas y aplicar la migración inicial para crear la base de datos de identidad.

# Instalar herramientas de EF si no las tienes
dotnet tool install --global dotnet-ef

# Agregar migración inicial especificando el directorio de salida
dotnet ef migrations add InitialCreate -o Data/Migrations

# Actualizar la base de datos
dotnet ef database update

El parámetro -o Data/Migrations indica a EF Core dónde debe guardar los archivos de la migración para mantener el proyecto organizado.

Nota: Por defecto, el proyecto utilizará SQLite. Si deseas usar SQL Server, deberás cambiar la cadena de conexión en appsettings.json y cambiar el proveedor en Program.cs.

3 Ejecución y Verificación

Finalmente, iniciamos la aplicación para verificar que el sistema de autenticación y los componentes interactivos funcionan correctamente.

dotnet watch run

Al abrir la aplicación en el navegador, verás los enlaces de Register y Login en la parte superior derecha, lo cual confirma que Identity se configuró exitosamente.

Resumen Tecnológico

Blazor Server

Interactividad en tiempo real gestionada mediante SignalR desde el servidor.

EF Core

Mapeo objeto-relacional para interactuar con la base de datos usando C#.

Identity

Sistema completo de seguridad para manejo de usuarios, perfiles y roles.
